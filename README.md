# SalesEvent – Full Stack E-Commerce Platform

## 🚀 Tech Stack
- Frontend: React.js (Vite)
- Backend: Spring Boot
- Database: MySQL

## 📦 Project Structure
backend/ → Spring Boot REST APIs  
frontend/ → React UI  

## 🔑 Features
- JWT Authentication
- Role-based access (Admin/User)
- Product & Order Management
- Analytics Dashboard

## ▶️ Run Locally

### Backend
cd backend  
mvn spring-boot:run  

### Frontend
cd frontend  
npm install  
npm run dev
